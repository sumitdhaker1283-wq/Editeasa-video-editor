EditEase Render-ready package
=============================
This package includes a Node.js backend (server.js) and a static frontend in /public.
To run locally:
  1. npm install
  2. npm start
  3. Open http://localhost:4000

To deploy on Render.com:
 - Create a new Web Service -> Connect repo or upload
 - Build command: npm install
 - Start command: npm start
 - Render will run the server and serve the frontend from /public
